Was ist das Marktstammdatenregister?

Das Marktstammdatenregister ist das Register für den deutschen Strom- und Gasmarkt. Es wird von der Bundesnetzagentur geführt.

Alle Anlagenbetreiber, Netzbetreiber, Energielieferanten sowie alle Strom- und Gaserzeugungsanlagen sind darin registriert.

Das macht es zu einer wichtigen Datenquelle, um den Fortschritt der Energiewende nachzuvollziehen.

Was ist Marktstammdatenregister.dev?

Das Ziel von Marktstammdatenregister.dev ist es, die Einschränkungen der offiziellen Webseite zu beseitigen.

Daten können beliebig gefiltert und sortiert werden. Eine Kartenansicht und ein rudimentäres Werkzeug zur graphischen Darstellung der Ergebnisse stehen zur Verfügung.

Wer sich mit SQL auskennt, kann noch flexibler Daten kombinieren.
